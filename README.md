# Weight Converter 

A weight conversion program built in Java.

## About
The user chooses to convert either lbs to kg 
or kg to lbs. The program then takes their 
weight and converts it using the correct formula.

## What I learned
- The formula for converting lbs to kg is x 0.453592
- The formula for converting kg to lbs is x 2.20462
- When to use double vs int:
  - int is for whole numbers like a menu choice (1 or 2)
  - double is for decimal numbers like weight (72.5)
